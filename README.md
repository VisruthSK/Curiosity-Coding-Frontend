# Curiosity Coding Site

Vibe coded, browser-only CSV coding internal research tool for coding [Curiosity](https://www.visruth.com/posters/USCOTS%20Curiosity%20Poster.svg) (see us at eCOTS, ICOTS, etc. too!) student responses. Everything is, of course, done in browser--there's no backend, no data ever leaves your laptop (autosaves and all that into `localstorage`).
